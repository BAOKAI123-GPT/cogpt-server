# 翰文上线清单（大白话版）

翰文已作为 CoGPT 后端的"第二个产品"接好。**支付、短信、账号体系全部复用 CoGPT 现成的**，所以要做的事很少。

## 一、把后端更新一下（在 cogpt.art 那台服务器上）
> CoGPT 既然能用，说明这台服务器和它的 .env（里面有 zpay 商户号、阿里云短信 key、JWT 等）都早就配好了。我们只是更新代码。

1. 把更新后的 `cogpt-server` 整个目录放到那台服务器（覆盖旧的，或 `git pull`）。
2. 在该目录里执行一条命令：
   ```bash
   bash deploy.sh
   ```
   （它会自动：备份数据库 → 装依赖 → 同步数据库结构 → 构建 → 重启）

就这一步。如果你不懂服务器，把这条命令发给当初帮你搭 cogpt.art 的人，2 分钟搞定。

## 二、后台填几项（点网页就行，不碰代码）
打开 `你的域名/console-7kq9mx2p`，用管理员账号登录，进「配置」页，拉到 **「翰文（文书）配置」** 那一栏：

| 要填的 | 填什么 |
|---|---|
| 翰文·中转站地址 | `https://api.qingyuntop.top`（默认已填） |
| **翰文·中转站 API Key** | 你的青云 key（见项目总结文档，不入库）（**必填**，这是翰文 AI 的钥匙） |
| 翰文·对话模型 | `claude-sonnet-4-6`（默认已填） |
| 翰文·Windows/macOS/安卓 下载直链 | 软件安装包的下载地址（GitHub Release 镜像，见下） |

套餐价格默认已是 **基础 ¥888 / 升级 ¥1888 / 至尊 ¥3888**、周 token 额度也设好了，想改就在「翰文套餐」那几行改。填完点「保存配置」。

## 三、软件安装包放哪供下载
翰文的 Win/Mac 安装包传到一个能直链下载的地方（推荐 GitHub Release + 国内镜像），把链接填进第二步的"下载直链"。官网下载页会自动读这些链接。

## 完成后
- 用户打开翰文 App → 手机号收验证码登录 → 按套餐用（额度每周刷新）→ 额度用完在 App 内扫码充值/升级。
- 后台「用户」页能看每个用户的翰文套餐/剩余 token，也能手动开通/加额度。
- 后台「订单」页能看翰文/生图的付款记录。

> 真正需要你账号的只有：青云 key（已给）、那台服务器、zpay/阿里云（CoGPT 已在用）。其余我都做好了。
